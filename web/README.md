This React application is the **web interface** of the Digital Diploma
Generation and Verification System.

It is used by:
- Administrators to manage students and generate diplomas
- Public users to **verify diploma authenticity** via QR code or UUID
- Institutions to validate signed PDF diplomas

The app consumes a **secured Django REST API** shared with the Flutter mobile app.

---

## Features
-  Diploma generation dashboard
-  Public diploma verification (QR / UUID)
-  Signed PDF verification
-  JWT authentication
-  Environment-based API configuration
-  Verification statistics dashboard

---

## Backend
The backend is a Django REST API.
